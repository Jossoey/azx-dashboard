**PT. AZX Car Management & Sales Dashboard**
A full-stack web application for displaying car sales data, car inventory data, and AI-chatbot for car recommendation.

**Built with:**
Frontend: React + Recharts
Backend: Node.js + Express
Database: PostgreSQL
Deployment Ready: AWS Elastic Beanstalk compatible

**Features:**
(1) Sales Dashboard
    - Yearly unit sales chart
    - Monthly unit sales breakdown
    - YoY revenue growth calculation
    - Revenue summary display
(2) Inventory Dashboard
    - Stock tracking by model
    - Stock tracking by location
(3) AI-Chatbot for Car Recommendation
    Recommends best car based on:
      - Family size
      - Desired features
      - Size preference (optional)
      - Budget (optional)
